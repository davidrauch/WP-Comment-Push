== Description ==

WP Comment Push sends every new Comment to your Boxcar account, resulting in a Push Notification on your iPhone/iPad.
You need a BoxCar account to use this plugin!

== Installation ==

1. Upload `wp-comment-push` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add "Growl/API" as a Service in your BoxCar app
4. Input your Boxcar account details in the plugin's settings
